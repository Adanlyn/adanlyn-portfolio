✦ Projeto Destaque

TalkClass

Sistema web completo para feedback e automação de dados educacionais. Integra IA generativa via Gemini para análise de dados e extração de insights para gestores, com arquitetura de microserviços.

.NET Node.js Python FastAPI React PostgreSQL Docker JWT Gemini AI
  • Microserviço Python/FastAPI integrado à API Gemini para análise e extração de insights automáticos
  • APIs REST seguras com autenticação JWT e persistência em PostgreSQL
  • Dashboards dinâmicos em React/TypeScript para visualização de métricas
  • Orquestração de containers com Docker para portabilidade e manutenção
  • Arquitetura de microserviços com separação clara de responsabilidades
