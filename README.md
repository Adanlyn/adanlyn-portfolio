## 🔎 Sobre o projeto

O portfólio foi construído utilizando HTML, CSS e JavaScript puro, priorizando:

- Estrutura semântica
- Responsividade
- Organização de código
- Separação clara de responsabilidades
- Performance e leveza (sem frameworks)

Também implementei interações com IntersectionObserver para animações baseadas em scroll e navegação ativa dinâmica.
